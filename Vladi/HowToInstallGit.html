How to Install Github and Git

step by step guide for begineners

important Clarification

note: GitHub is cloud-based hosting service (a website) where you can write and store your code,while Git is the local command-line tool (software) you install on your computer to track changes.To use GitHub, you must first install Git.

Step 1: Installing Git

Choose your operating system (Windows, macOS, or Linux) and follow the instructions below:

Windows installation

The fastest way is using the terminal(PowerShell or Command Prompt):

winget install Git.Git

Alternatively, you can download the installer from the official website and follow the setup wizard.

Go to official Git website

macOS Installation

Open Terminal and type the following command to check if you already have it:

git --version

If you prefer using Homebrew, run:

brew install git

Linux Installation

Run the following command in your terminal:

sudo apt update

sudo apt install git

Step 2: installing configuration

Once the installation is complete, open a new terminal window and configure your username and email:

git config --global user.name "Your Name"

git config --global user.email "your.email@example.com"

Step 3: Create a GitHub Account

Go to the GitHub website

Go to GitHub.com

Click on the "Sign up" button in the top right corner

Enter your email,create a password, and choose a username

Verify your account through the link sent to your email
